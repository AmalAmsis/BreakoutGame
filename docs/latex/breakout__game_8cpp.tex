\doxysubsection{Detailed Description}
Implementation of systems and entity creation functions using BAGEL ECS. 

This file contains the definitions of all systems and entity constructors used in the Breakout-\/style game, implemented based on the ECS model with BAGEL. All logic is placeholder only; the focus is on structure and mask-\/based iteration.
